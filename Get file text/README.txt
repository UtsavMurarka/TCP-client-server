TCP Client-Server model
------------------------

STEPS TO RUN:
	1. Compile server.c by running : gcc -o server server.c
	2. Compile client.c by running : gcc -o client client.c
	3. Open two separate terminals, one from the Server directory, and other from the Client directory
	4. Run ./server in the server terminal
	5. The program will ask for port number for server to be hosted, enter valid port number.
	6. Run ./client in the client terminal
	7. The program will ask for IP, Enter 127.0.0.7 IP (for localhost)
	8. The program will ask for port number, enter the same port number used for server
	9. Enter filename to be queried
